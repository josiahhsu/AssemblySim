Assembly Simulator

You can write programs in assembly code below. The syntax for each line is [instruction] [Operand1] [Operand2] [...]. Empty lines will be treated as no-ops and skipped. Lines starting with a pound sign will be treated as comments and skipped. Operands can either be an immediate value (prefixed with $) or a register name (prefixed with %). Operand values and operation results are limited to 32-bit signed integers, and immediate values must also be positive. By convention the return value will be stored in register %rax. Register %rsp is the stack pointer, meaning the stack top is stored at the position specified by %rsp. Registers %rdi, %rsi, %rdx, %rcx, %r8, and %r9 can have their values set as input arguments.
